## E-TICKET CMV ANALYSIS

### Reference (From guides/09-ecticket.md)

**E-Ticket Formula (Different from AIO):**
`
CheckMacValue = SHA256( toLowerCase( URLEncode( HashKey + Data明文 + HashIV ) ) )
`

**Steps:**
1. Take Data plaintext (before AES encryption)
2. Concatenate: HashKey + Data + HashIV (NO parameter sorting!)
3. URLEncode (with .NET replacements)
4. toLowerCase
5. SHA256
6. UPPERCASE

**Key Differences from AIO:**
- AIO: Sort parameters individually, add = and & separators
- E-Ticket: Take entire JSON plaintext as-is, no sorting
